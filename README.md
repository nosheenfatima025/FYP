# Final Year Project (PARKIFY) – Smart Vehicle Managemen System

##  Project Overview
This is a Final Year Project developed to automate and manage vehicle entry, exit, and user account activities in a smart and efficient way.

The system is designed to provide a secure and digital solution for tracking vehicles, managing user accounts, and maintaining complete history records with notifications.

---

##  Objective
The main objective of this project is to:
- Automate vehicle entry and exit tracking
- Provide user account management system
- Maintain secure and digital records
- Reduce manual workload and errors
- Improve efficiency through a web/mobile-based system

---

## ⚙️ Features

### 👤 User Module
- User Registration & Login
- Personal Account Dashboard
- Vehicle Registration
- Balance / Account Management

### 🚗 Vehicle Module
- Vehicle Entry Tracking
- Vehicle Exit Tracking
- Complete Vehicle History

### 🔔 Notification System
- Entry/Exit Alerts
- Account Activity Notifications

### 🛠 Admin Panel
- Manage Users
- View Reports
- Monitor Vehicle Activity
